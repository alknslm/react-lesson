JsonServer ile fake api kullanacağız.

npm install -g json server diyip kurulum yapıyoruz. (-g koymayınca version görünmedi)
src altında db.json dosyası oluşturuyoruz
json-server --watch .\src\db.json --port 4000 komutuyla istediğimiz portta çalıştıabiliriz.
